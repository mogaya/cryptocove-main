# Globpay

Globpay is a mock crypto gateway experience that showcases how businesses can accept, convert, and settle digital currency payments in real time. It highlights live market data, merchant tooling, and onboarding flows for a global-ready payment infrastructure. Built with Next.js, React, TypeScript, and Tailwind CSS.

## Development

Follow these steps to run the project locally.

1. Install dependencies

```sh
npm install
```

2. Get Your Free API Key at [CoinGecko API](https://www.coingecko.com/en/api)

3. Create a <code>.env.local</code> file in the root directory.

4. Add: <code>NEXT_PUBLIC_COINGECKO_API_KEY=YOUR_API_KEY</code> and make sure to replace <code>YOUR_API_KEY</code> with your own key.

5. Run the project

```sh
npm run dev
```
